Databag API

The Databag API uses requests from users to share their personal data and host group conversations in a decentralized server. The Databag API allows users to self-host their own apps while maximizing the communication of developers.

About the Databag API

Databag’s API is an open network that provides a decentralized approach for messaging their peers, colleagues, friends, and family members. The API offers the user a way to store their personal data without the concern of data collection and to host group conversations without the intrusion of the user’s privacy.

The API is written in JavaScript and the responses will be formatted in JSON but by using the Swagger UI, Databag will yield coding samples for several programming languages that are mentioned in this reference guide.
